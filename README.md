# Laboratorio di simulazione numerica

##### Università degli Studi di Milano, A.A. 2018-2019


### Esercizi
Questa repository contiene i file sorgenti dei programmi scritti per gli esercizi del corso di Laboratorio di Simulazione Numerica dell'anno accademico 2018-2019 all'Università degli Studi di Milano.
In ogni cartella si trovano i file relativi agli esercizi di ogni lezione, cioè i sorgenti in C++ con un `makefile` per compilarli e un notebook di `jupyter` per presentare, anche graficamente, i risultati con Python.
